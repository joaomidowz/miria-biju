# Miriã Biju Landing Page

Site estático da Miriã Biju, seguindo o padrão das landing pages em `frontend-projects`.

## Arquivos

- `index.html`: estrutura da página e conteúdo.
- `style.css`: layout, responsividade e identidade visual.
- `script.js`: menu mobile, animações, carrossel e seletor de lojas.
- `images/logo-miria-biju.svg`: logo em SVG baseado na referência enviada.

## Como visualizar

Abra `index.html` no navegador.

## Imagens do carrossel

Os espaços de acessórios usam proporção `1:1` e estão preparados para receber de 9 a 18 imagens. Quando as fotos finais existirem, substitua cada bloco `.accessory-photo` por uma imagem quadrada.
